    CLI Reference¶

    This page provides documentation for our command line tools.

    rohe-cli¶

    CLI group for ROHE commands.

    Usage:

    rohe-cli [OPTIONS] COMMAND [ARGS]...

    Options:

    Name Type Description Default
    --version boolean Show the version and exit. False
    --help boolean Show this message and exit. False

    Subcommands

    • observation: Group for observation related commands.
    • orchestration: Group for orchestration related commands.
    • start: Group for starting services.

    rohe-cli observation delete-app¶

    Delete an application from the observation service.

    Usage:

    rohe-cli observation delete-app [OPTIONS]

    Options:

    Name Type Description Default
    --app text Application name dummy
    --run text Experiment name experiment1
    --user text User name aaltosea1
    --url text Registration URL http://localhost:5010/agent/delete
    --help boolean Show this message and exit. False
    rohe-cli observation register-app¶

    Register an application with the observation service.

    Usage:

    rohe-cli observation register-app [OPTIONS]

    Options:

    Name Type Description Default
    --app text Application name test
    --run text Experiment name/id experiment2
    --user text User name aaltosea1
    --url text Registration URL http://localhost:5010/registration
    --output_dir text Output directory. Ensure the directory exists. None
    --help boolean Show this message and exit. False

    rohe-cli start observation-service¶

    Start the observation service.

    Usage:

    rohe-cli start observation-service [OPTIONS]

    Options:

    Name Type Description Default
    --debug boolean Run the Flask development server if set, otherwise run Gunicorn False
    -h, --host text Host to bind the server to 0.0.0.0
    -p, --port integer Port to bind the server to 5010
    --help boolean Show this message and exit. False
